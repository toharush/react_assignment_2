# react_assignment_2

## Installation
Working Cloud based or local
### DATABSE
  [mongodb](https://fastdl.mongodb.org/windows/mongodb-windows-x86_64-6.0.3-signed.msi)
  
```env
  PORT=NUMBER
  MONGO_URI=URI_CLOUD_OR_LOCAL
```
  
## Starting the server

```nodejs
# move to server folder
cd server

# install dependencies
npm install

# start the server
npm start
```
## Starting the Client

```nodejs
# move to client folder
cd client

# install dependencies
npm install

# start the client
npm start
```
